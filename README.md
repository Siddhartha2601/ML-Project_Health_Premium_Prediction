# Health Insurance Premium Prediction

This project predicts health insurance premium using machine learning.

The model uses age and health related data to predict how much insurance premium a person may pay.

---

## Project Files

- `main.py` – Streamlit app file  
- `prediction_helper.py` – Loads model and makes prediction  
- `requirements.txt` – List of required Python packages  
- `artifacts/` – Contains trained model and scaler files  
- `.gitignore` – Ignores unnecessary files  

---

## Requirements

You need Python installed on your system.

Install all required libraries using this command:

```bash
pip install -r requirements.txt
